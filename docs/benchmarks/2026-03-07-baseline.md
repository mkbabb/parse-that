| sonic-rs | 2415 | 1947 | 2599 | 3112 | 1576 | 2859 |

### Work equivalence notes
- parse_that, nom, winnow, pest: zero-copy borrowed `&str`, raw escapes
- serde_json: owned `String`, fully decoded escapes (MORE work)
- serde_json_borrow: `Cow<str>`, decoded escapes
- jiter: `Cow<str>`, selectively decoded
- simd-json: `Cow<str>` borrowed mode, `.to_vec()` per iteration (mutable input)
- sonic-rs: owned `Value`, arena alloc, fully decoded (MORE work, SIMD-accelerated)

### Summary
parse_that is **1.5-2.0x faster than nom/winnow** (work-equivalent peers) and within **0.7-0.85x of jiter** (which does slightly more work). SIMD parsers (simd-json, sonic-rs) are 1.5-3x faster but use platform-specific SIMD instructions.

## Rust CSS (MB/s)

| Parser | normalize (6KB) | bootstrap (281KB) | tailwind (3.6MB) |
|---|---|---|---|
| **parse_that** (L1.5) | **38** | **6** | N/A (too slow) |
| cssparser (L0-L1) | 690 | 446 | 275 |
| lightningcss (L2) | 239 | 113 | FAIL (parse error) |

### Notes
- parse_that CSS is pure combinator composition — no monolithic scanners or SpanParser optimization yet
- cssparser is servo's tokenizer with callback visitors (less work than parse_that)
- lightningcss does more work (L2 semantic parse) but fails on synthetic tailwind file
- parse_that CSS parser needs SpanParser leaf treatment to be competitive (same optimization that accelerated JSON from ~100 MB/s to ~1000 MB/s)

## TypeScript CSS

| Parser | normalize (6KB) | bootstrap (274KB) |
|---|---|---|
| **parse-that** (L1.5) | **1.00x** | **1.00x** |
| postcss (L1) | 3.87x slower | 3.36x slower |
| css-tree (L1-L2) | 5.69x slower | 4.10x slower |

### Fairness validation
- Declaration counts match exactly: normalize 57/57, bootstrap 5542/5542
- parse-that produces typed values (`{type: "dimension", value: 100, unit: "px"}`) while postcss returns raw strings (`"100px"`)
- parse-that produces typed selectors (compound/complex AST) while postcss returns raw strings
- parse-that does MORE semantic work than postcss — the 3.4-3.9x advantage is conservative
